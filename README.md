# ProfInjector
Nice injector with Professionall Functions
A lightweight, educational DLL injector for Windows. This tool demonstrates the use of Windows API functions to inject a dynamic-link library (DLL) into a running process. It is intended for learning purposes, debugging, and system programming research.
Features

 #   Inject a DLL into a process by process ID.

 #   Inject a DLL into a process by process name (resolves to PID).

 #   Support for both 32-bit and 64-bit processes (the injector's architecture must match the target process).

 #   Simple command-line interface.

 #   Error handling with descriptive messages.
